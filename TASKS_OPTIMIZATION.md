# 任务调度优化说明

## 优化概览

对任务调度页面进行了全面优化，提升了用户体验和功能完整性。

## 实现的功能

### 1. 任务列表筛选优化

#### ✅ 仅显示有日志开关
- 位置：筛选区第二行右侧
- 默认状态：开启
- 功能：过滤掉没有日志的任务记录
- 可以关闭以显示所有任务

#### ✅ 时间筛选器
- 类型：日期时间范围选择器
- 筛选依据：Trace 记录进入数据库的时间（created_at）
- 格式：YYYY-MM-DD HH:mm:ss
- 支持快捷选择和自定义范围

#### ✅ 日志关键字过滤
- 位置：筛选区第二行
- 前置条件：需要先选择任务类型
- 功能：根据日志内容关键字过滤任务
- 未选择任务类型时自动禁用

#### ✅ 分页功能
- 每页数量：10/20/50/100 可选
- 默认：20 条/页
- 排序：按 Trace 创建时间降序（最新的在前）
- 包含总数显示和页码跳转

### 2. 任务详情优化

#### ✅ 对话框形式展示
- 从 Drawer（抽屉）改为 Dialog（对话框）
- 宽度：680px，更适合居中展示
- 使用描述列表组件美化展示

#### ✅ 字段格式化
- 自动识别并美化字段标签（中文显示）
- JSON 数据自动格式化，带缩进
- 空值统一显示为 "-"
- 长内容支持滚动查看

### 3. 日志查看优化

#### ✅ 系统类型过滤
- 选项：Python / Java
- 独立筛选，不影响其他条件

#### ✅ 时间过滤
- 类型：日期时间范围选择器
- 不级联，独立使用
- 格式：YYYY-MM-DD HH:mm:ss

#### ✅ 关键字搜索
- 实时搜索日志内容
- 支持模糊匹配
- 带搜索图标提示

#### ✅ 时间线逆序显示
- 最新的日志显示在最前面
- 便于快速查看最新状态

#### ✅ 固定间隔自动刷新
- 刷新间隔：5 秒
- 增量更新，不影响当前查看位置
- 不全局刷新页面
- 保持逆序排列
- 关闭抽屉时自动停止刷新

### 4. 样式优化

#### 筛选区域
- 使用浅蓝色背景 (#f8faff)
- 圆角边框，视觉层次清晰
- 控件对齐优化，间距统一
- 响应式布局，移动端自动换行

#### 日志展示
- 卡片式设计，每条日志独立展示
- Hover 效果：高亮边框和阴影
- 时间、级别、来源用 Tag 标签区分
- 消息内容带左侧蓝色边框
- 浅灰背景提升可读性

#### 表格优化
- Trace ID 使用等宽字体
- 状态标签使用 light 效果
- 操作列固定在右侧
- 响应式适配

#### 响应式设计
- 移动端：筛选器自动全宽显示
- 平板：布局自动调整
- 桌面：最佳显示效果

## 后端 API 要求

为确保功能完整，后端需要支持以下接口参数：

### GET /system/tasks
```
参数:
- page: 页码
- pageSize: 每页数量
- status: 状态筛选
- taskType: 任务类型
- triggerEntry: 触发入口
- logKeyword: 日志关键字（需要 JOIN 日志表）
- onlyWithLogs: 是否只返回有日志的任务（布尔值）
- startTime: 开始时间（created_at）
- endTime: 结束时间（created_at）

返回:
{
  records: [...],  // 任务列表
  total: 100      // 总数
}

排序: created_at DESC（最新的在前）
```

### GET /system/tasks/:traceId/logs
```
参数:
- systemType: 系统类型（python/java）
- startTime: 开始时间（logged_at）
- endTime: 结束时间（logged_at）
- keyword: 关键字搜索

返回: 日志数组

注意: 前端会自动逆序，后端可以按 logged_at ASC 返回
```

## 使用说明

1. **查询任务**：设置筛选条件后点击"查询"按钮
2. **重置筛选**：点击"重置"按钮清空所有筛选条件
3. **查看详情**：点击"详情"按钮在对话框中查看任务完整信息
4. **查看日志**：点击"日志"按钮打开日志抽屉，自动开始刷新
5. **筛选日志**：在日志抽屉中设置系统类型、时间范围或关键字后点击"筛选"
6. **关闭日志**：关闭抽屉时自动停止刷新

## 注意事项

1. 日志关键字过滤需要先选择任务类型，否则该功能会被禁用
2. 日志自动刷新每 5 秒执行一次，关闭抽屉会自动停止
3. 分页切换会重新加载数据
4. 所有时间筛选器都支持快捷选择和自定义输入
